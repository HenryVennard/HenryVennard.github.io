Laser Guitar

Solidworks Project

This Website

It's like a guitar. Except it has lasers. That was our idea with the laser guitar. My team and I, designed and built a toy that mimiced an acoustic guitar but used lasers and photodiodes with an arduino to make sound. It was a really fun project and I learned a lot about 3D printing, ardunios and soldering. We ended up winning "Best in Class Award" for being the highest rated project out of 5 other teams.

The final project of my solidworks class was to model an object with 3 moving parts. I naively chose a grigri, a climbing belay device. It has an almost organic shape and was more difficult than I imagined to model. I learned a lot about solidworks in the progress and ended with a 100% on the project.

This website has been a fun project. I wanted a personal website to let people know a bit about who I am and what I've done. I thought about using a template website but decided the best way would be to code it myself. In the process I've learned a lot about html and css. I even managed to dive into javascript and make things a bit more dynamic.
